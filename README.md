 🏡 StayScope Airbnb Market Insights Dashboard

 📌 Project Overview

StayScope is an interactive Tableau dashboard designed to analyze and visualize Airbnb listing trends across the globe. The project examines **12,805 Airbnb listings spanning 119 countries**, providing valuable insights into pricing patterns, listing distribution, accommodation characteristics, and customer ratings. The dashboard enables users to explore market trends and identify key factors influencing Airbnb pricing and performance across different regions.



 🛠️ Tools Used

* **Tableau Public** – Data visualization and dashboard development
* **Microsoft Excel / CSV Dataset** – Data storage and preparation
* **Data Analytics Techniques** – Data cleaning, aggregation, and exploratory analysis



## 📊 Dataset Information

* **Dataset Type:** Global Airbnb Listings
* **Total Listings:** 12,805
* **Countries Covered:** 119
* **Key Attributes:**

  * Country
  * Average Price
  * Rating
  * Number of Beds
  * Listing Count
  * Property Information

The dataset provides a comprehensive overview of Airbnb accommodations worldwide, enabling comparative analysis across countries and property characteristics.

---

## 📈 Visualizations

### 1. Average Price by Country

Displays the average Airbnb listing price across different countries, helping identify premium and budget-friendly markets.

### 2. Rating Distribution

Shows the distribution of customer ratings, highlighting overall guest satisfaction trends.

### 3. Listings Count by Country

Visualizes the number of Airbnb listings available in each country to identify the most active markets.

### 4. Beds vs Average Price

Analyzes the relationship between the number of beds and listing prices to understand accommodation pricing patterns.

### 5. Rating per Country

Compares average ratings across countries, providing insights into customer experience and service quality.

---

## 🔍 Key Insights

* 🇭🇳 **Honduras recorded the highest average Airbnb price** among all countries analyzed.
* 🇮🇳 **India had the highest number of Airbnb listings**, making it the most represented country in the dataset.
* 🛏️ **Properties with a higher bed count generally command higher prices**, indicating a positive relationship between accommodation capacity and pricing.
* ⭐ Customer ratings showed variation across countries, reflecting differences in guest experiences and service standards.
* 🌍 Listing distribution is uneven across countries, with some regions demonstrating significantly greater Airbnb market activity.
